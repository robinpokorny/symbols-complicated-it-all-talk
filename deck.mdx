## symbol vs enumerable

---

```js
const c = Symbol();
const d = Symbol.for("d")

const o = { a: 1, [c]: 10 };

Object.defineProperty(o, "b", {
  value: 2,
  enumerable: false
});
Object.defineProperty(o, d, {
  value: 20,
  enumerable: false
});
```
